Tingyo Chang // Assignment 08: Final Project Proposal

Topic:

I'd like to create an interactive writing portfolio that showcases both my writing and design skills. The goal would be to create an easy access location for all of my creative works to be easily accessible.

Audience:

My ideal audience would be future employers or possible clients that may want to access or review my writing. The goal is to create a website that is appealing to any potential clients.

Design and Style:

Color scheme:

I want the focus to be the content, so the color scheme will be mainly focused around white, black, and a very light grey to draw focus toward the website elements.

Color palette:

Primary: #FFFFFF Secondary: #000000 Tertiary: #D3D3D3

Inspiration:

Kristen Bateman Alice Driver Tara Pixley

Scope:

In total, there will be 4 pages: A landing page/homepage, Creative Writing, Analytical, and Contact Me page.

Extras:

One extra will be event-driven DOM manipulation. Users will be able to clear out the "Contact Me" fields. Another extra will be React-Bootstrap from React library. This will allow me to further format some of the photos that I'd like to use on my website.
